Installation

Follow the steps below to set up your project:

Setting up Flutter

  1. Download and install Flutter 3.19.5 - 3.19.5+ SDK from the official Flutter website or Flutter SDK archive .
  2. Extract the downloaded zip file and add the `flutter/bin` directory to your system's PATH.
  3. Verify the installation by running flutter doctor in the terminal. Follow any additional steps suggested.
  4. Open project on VS code or Android Studio.

Setting up Laravel

  1. Ensure you have Composer installed. If not, download it from the official Composer website.
  2. Install Laravel by running composer global require laravel/installer in the terminal.
  3. Create a new Laravel project by running laravel new project_name.
  4. Navigate to the project directory cd project_name and run php artisan serve to start the local development server.

Integrating Flutter with Laravel

  1. Set up API routes in Laravel to handle backend logic. Define routes in routes/api.php and create controllers as needed.
  2. In your Flutter project, add the http package to your pubspec.yaml file to handle API requests: 
    dependencies:
  http: ^0.13.3
  3. Run flutter pub get to install the package.
  4. Use the http package to make API requests to your Laravel backend from your Flutter app.

Firebase Setup

Follow the steps below to set up your project:

Create a new project in Firebase

Follow these steps to create a new project in Firebase:

  1. Go to the Firebase Console and sign in with your Google account.
  2. Click on the "Add project" button (if you don't have any existing projects) or select an existing project to add Firebase to it.
  3. Enter a project name and optionally customize the project ID.
  4. Click "Continue" and follow any additional prompts to create your Firebase project.

If you're new to Firebase, watch this tutorial to get started:

This tutorial covers the basics of creating a project in Firebase and integrating Firebase services into your applications.

Related information:

Create a new app in Firebase

Follow these steps to create a new app in Firebase and integrate it into your Flutter project for Android and iOS:

  1. Go to the Firebase Console and sign in with your Google account.
  2. Create a new Firebase project or select an existing one.
  3. Click on "Add app" to add a new app to your Firebase project.
  4. Choose the platform (Android or iOS).
  5. Follow the steps to register your app with Firebase.

Download the necessary configuration files for both Android and iOS:

  • For Android:
    1. Download the google-services.json file.
    2. Place the google-services.json file in the android/app directory of your Flutter project.
  • For iOS:
    1. Download the GoogleService-Info.plist file.
    2. Open your Flutter project in Xcode.
    3. Drag and drop the GoogleService-Info.plist file into the Runner/Runner directory in Xcode.

Firebase generate new private key

Follow these steps to generate a new private key in Firebase and update your backend Laravel project:

  1. Go to the Firebase Console and sign in with your Google account.
  2. Navigate to the "Project settings" by clicking on the gear icon next to your project name.
  3. In the "Service accounts" tab, click on "Generate new private key".
  4. Confirm the action and download the generated private key file.

If you need a detailed tutorial on generating a new private key in Firebase, watch this video:

Download the key and update your backend Laravel project:

  1. Locate the existing public/google-services.json file in your Laravel project.
  2. Replace the contents of the public/google-services.json file with the new data from the generated private key file.

Make sure to securely store the new private key and update any configurations that rely on it.
